# 第三章 社会主义改造理论

## 一、判断题

1. 中华人民共和国成立初期，建立社会主义国营经济的主要途径是剥夺地主阶级的土地和财产。（）
2. 中华人民共和国的成立，标志着中国进入了社会主义初级阶段。（）
3. 社会主义改造时期，农村的阶级政策是依靠贫下中农，发展互助合作，由逐步限制到最后消灭富农剥削。（）
4. 农业改造中的初级农业合作社具有社会主义萌芽的性质。（）
5. “定股定息”是指企业的利润按国家所得税、企业公积金、工人福利费和资方红利这四个方面进行分配。（）
6. 党在过渡时期总路线最显著的特点是实行社会主义现代化和社会主义改造同时并举。（）
7. 中国民族资产阶级在社会主义革命时期仍具有两面性，既有剥削工人取得利润的一面，又有拥护中国共产党领导、拥护宪法、愿意接受社会主义改造的一面。（）
8. 新民主主义社会是一个独立的社会形态，是由新民主主义向社会主义转变的过渡性社会形态。（）
9. 新民主主义社会时期，国家资本主义经济是私人资本主义经济向社会主义国营经济过渡的形式。（）
10. 1954年9月，第一部《中华人民共和国宪法》颁布实施，这部宪法明确规定了我国人民代表大会制度的国体和人民民主专政的政体。（）
