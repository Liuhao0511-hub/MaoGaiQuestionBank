# 第二章 新民主主义革命理论

## 一、判断题

1. 解决中国革命问题的基本前提是建立工农联盟。（）
2. 鸦片战争后，由于西方列强的入侵和封建统治腐败，中国逐步成为半殖民地半封建社会，这是近代中国最基本的国情。（）
3. 在半殖民地半封建的近代中国，占支配地位的主要矛盾是帝国主义和中华民族的矛盾、封建主义和人民大众的矛盾。（）
4. 官僚资本主义是依靠帝国主义、勾结封建势力、利用国家政权力量而发展起来的买办的封建的国家垄断资本主义。（）
5. 在十月革命的影响下，以五四运动为标志，中国无产阶级开始作为独立的政治力量登上历史舞台，成为革命的领导力量。（）
6. 新民主主义革命实践，是新民主主义革命理论得以形成的基础和源泉。（）
7. 毛泽东在《中国革命和中国共产党》中完整地总结和概括了新民主主义革命总路线的内容。（）
8. 中国革命的对象是帝国主义、封建主义和官僚资本主义，它们是压在中国人民头上的三座大山。（）
9. 农民是中国革命最基本的动力。（）
10. 新民主主义革命性质是无产阶级领导的社会主义革命。（）
11. 中国革命必须走农村包围城市、武装夺取政权的道路，是由中国所处的时代特点和具体国情决定的。（）
12. 新民主主义革命的三大法宝是实事求是、统一战线、党的建设。（）
13. 无产阶级及其政党对中国革命的领导权是自然而然得来的。（）
14. 完成新民主主义革命的伟大胜利极大改变了世界政治格局，鼓舞了被压迫民族和被压迫人民争取解放斗争。（）
15. 新民主主义革命理论，是以毛泽东同志为主要代表的中国共产党人，把马克思列宁主义基本原理同中国革命具体实践相结合，在认真总结中国革命实践经验基础上形成的具有独创性的革命理论。（）
