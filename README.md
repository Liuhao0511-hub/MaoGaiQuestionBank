# 毛概题库练习系统

《毛泽东思想和中国特色社会主义理论体系概论》（2023 版）刷题练习平台，纯静态网站，双击即用。

> ⚠️ 题库以当年济南大学给出的题库为准，本仓库仅供参考。教材后续如有改版，题库可能不再适用。

## ✨ 功能

- **10 章 · 451 题** — 导论至结束语全覆盖，判断 / 单选 / 多选三种题型
- **答题进度** — 作答自动保存浏览器本地，刷新不丢失，支持 JSON 导入导出
- **错题本** — 按章节筛选错题，可跳回原题重做
- **题目解析** — 每道题均可点开查看解析，无需先作答
- **收藏夹** -选择想要收藏的题目放到收藏夹
- **暗色模式** — 系统跟随 + 手动切换，偏好持久化
- **响应式** — 手机 / 平板 / 桌面全适配

## 🚀 使用方式

```
双击 Web/index.html 即可
```

或部署到任意静态服务器（Nginx / GitHub Pages / Vercel）。

